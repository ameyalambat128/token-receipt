## Default post

I built a small local tool called Token Receipt.

It reads recent Codex and Claude Code session logs, computes a few defensible usage signals, and turns them into a satirical AI bill with a thermal-paper receipt image and share-ready copy.

The important constraint was keeping it local and making the skill flow work inside the tools people already use. The parser and renderer run locally, and the host agent writes the final roast using the session you are already in.
